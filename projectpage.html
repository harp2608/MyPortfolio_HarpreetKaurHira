Full-Stack Web Application for Train Accident Prevention

A comprehensive system using real-time data processing to detect and mitigate train accidents

2024 Individual Project

This project implements an end-to-end solution for real-time train monitoring and accident prevention. The system utilizes sensor data from trains and tracks to detect potential hazards and alert operators before accidents occur.

The application features a responsive React front-end for visualization and a Django backend for data processing and API services. AWS Lambda functions handle the serverless data processing workflows.

Key Features

  • Real-time data processing and anomaly detection
  • Interactive dashboard for monitoring train movements
  • Automated alert system for potential hazards
  • Historical data analysis for predictive maintenance
  • Secure authentication and role-based access control

Technologies Used

Django React AWS Lambda PostgreSQL Redis Docker WebSockets

End-to-End Integration Testing for Holiday Shopping

Automated test suite using Selenium WebDriver for Amazon's holiday shopping workflow

2023 Amazon Project

This project involved developing an automated test suite for Amazon's holiday shopping workflow to ensure smooth functionality during peak shopping periods. The solution automatically tests the entire shopping process from product search to checkout.

The implementation uses Selenium WebDriver with JUnit and TestNG frameworks to generate comprehensive test reports with screenshots and logs for tracking execution and identifying failure points.

Key Features

  • Cross-browser testing support (Chrome, Firefox, Safari)
  • Parallel test execution for faster results
  • Detailed test reports with screenshots
  • Automated failure detection and logging
  • CI/CD integration for continuous testing

Selenium WebDriver Java JUnit TestNG Maven Jenkins

SQL-Based Gurudwara Duties and Operations Management System

A relational database system for managing operations and duty assignments

2023 Academic Project

This project involved designing and implementing a comprehensive relational database system to efficiently manage daily operations and duty assignments for a Gurudwara (Sikh temple). The system tracks volunteer schedules, resource allocation, and event management.

  • User-friendly interface for data entry and retrieval
